# Sentix: Local-first, Security-Focused AI Agent

Sentix is an open-source AI agent designed for local operation, prioritizing security and privacy. It leverages local Large Language Models (LLMs) via Ollama and provides full agent capabilities accessed through a terminal interface, without relying on external cloud services or messaging platforms like WhatsApp.

## Features

-   **Local LLM Integration:** Powered by Ollama for running various local language models.
-   **Full Agent Capabilities:** Designed to observe, reason, and act within its environment.
-   **Terminal Interface:** All interactions happen via the command-line interface.
-   **Security Focused:** Employs sandboxing, detailed permission models, and comprehensive auditing.
-   **Offline First:** Operates entirely offline for core functionalities.
